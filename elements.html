The Computer Helper is a script that will give anyone help with using their computer. There are many different options in the computer helper, such as...

Chrome Cursor: Fix the invisible cursor gitch in Google Chrome Help Screen: Shows this screen

Isolate Programs: Close out of Windows to get maximum focus on a specific task

Clear Temp Files: Removes unnessecary files that bloat your system

View System Info: Shows all system information

Antivirus (WIP): Simple virus remover thats scans for common viruses, malwares, and security risks

Run on Startup: Run this program on startup

Refresh Explorer: Refreshes the Windows OS (so you don't have to restart)

Repair System: Repairs system files in case of corruption (will take a while)

Not Responding: Close all not responding tasks

File Organizer: Organizes files in your Desktop and Downloads folder

And Much More!

 

 

Colorblind Helper

The Colorblind Helper is a program that allows the user to just hover their cursor over a pixel on their screen and use an algorithm to convert the RGB value to a simple color name for the comon user to understand. Designers can also use this to obtain the RGB value of a pixel. Instructions to download the file is here.
